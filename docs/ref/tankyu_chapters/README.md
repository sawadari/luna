## 本書の特徴

### MBSE（モデルベースシステムズエンジニアリング）に焦点

本書は、現代のシステムズエンジニアリングのアプローチである**MBSE**を中心に解説しています:

- **「1枚のスライドのMBSE」**: MBSEの核心概念を1枚の図で理解
- **SysML**: システムモデリング言語の実践的活用
- **フレームワークとプロセス**: アプローチの体系的理解

## 主要な概念

### MBSEの核心

本書で解説される「1枚のスライドのMBSE」の5つの要素:

1. **アプローチ (Approach)**: フレームワーク・オントロジー・プロセスセット
2. **目標 (Goal)**: システム・モデル・ビュー
3. **視覚化 (Visualization)**: 表記法・ダイアグラム
4. **実装 (Implementation)**: ツール
5. **コンプライアンス (Compliance)**: ベストプラクティス・標準

### 重要な関係性

- モデルはシステムを抽象化する
- ビューはモデルを構成する
- ダイアグラムはビューを視覚化する
- 表記法（SysML等）はダイアグラムを定義する

## 関連標準

本書で参照される主要な標準:

- **ISO/IEC/IEEE 15288** - システムおよびソフトウェアエンジニアリング：システムライフサイクルプロセス
- **ISO/IEC/IEEE 42010** - システムズおよびソフトウェアエンジニアリング：アーキテクチャ記述
- **SysML (Systems Modeling Language)** - OMG標準のシステムモデリング言語
- **UML (Unified Modeling Language)** - オブジェクト指向モデリング言語
