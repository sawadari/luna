# Luna Simple App Example

シンプルなLunaアプリケーションの例です。

## 📋 概要

このサンプルは、Lunaフレームワークの基本的な使い方を示します。

## 🚀 実行方法

```bash
# ルートディレクトリから
npm install

# サンプルアプリを実行
cd examples/simple-app
npm run dev
```

## 📚 学べること

- Lunaの型システムの使用方法
- DEST理論の基本概念
- CrePS Boxの理解
- デプロイメント環境の設定
